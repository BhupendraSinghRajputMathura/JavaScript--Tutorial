1Ô∏è‚É£ setTimeout()

Definition:

The setTimeout() method is used to execute a function after a specified amount of time (in milliseconds). It runs the code only once after the delay.

Syntax:

setTimeout(function, delay, param1, param2, ...)

Parameters:

  • function: The function to be executed.
  • delay: Time in milliseconds before the function runs.
  • param1, param2, ...: Optional parameters to pass to the function.

Example 1: Basic Example


setTimeout(() => {
  console.log("Executed after 3 seconds!");
}, 3000);

Example 2: With Function Reference


function greet(name) {
  console.log("Hello, " + name);
}
setTimeout(greet, 2000, "Bhupendra Singh");
// Output after 2 seconds: Hello, Bhupendra Singh

2Ô∏è‚É£ clearTimeout()

Definition:

The clearTimeout() method is used to cancel a timeout that was previously set using setTimeout().

Syntax:

clearTimeout(timeoutID)

Example:


let timeoutID = setTimeout(() => {
  console.log("This will not run!");
}, 4000);

clearTimeout(timeoutID);
console.log("Timeout canceled!");
// Output: Timeout canceled!

3Ô∏è‚É£ setInterval()

Definition:

The setInterval() method executes a function repeatedly at specified intervals (in milliseconds) until it is stopped using clearInterval().

Syntax:

setInterval(function, interval, param1, param2, ...)

Parameters:

  • function: The function to be executed repeatedly.
  • interval: Time interval in milliseconds between executions.

Example 1: Printing Numbers Every 1 Second


let count = 1;
let intervalID = setInterval(() => {
  console.log("Count:", count);
  count++;
  if (count > 5) clearInterval(intervalID);
}, 1000);

// Output:
// Count: 1
// Count: 2
// Count: 3
// Count: 4
// Count: 5

Example 2: Displaying Time Every Second


setInterval(() => {
  let time = new Date();
  console.log("Current Time:", time.toLocaleTimeString());
}, 1000);

4Ô∏è‚É£ clearInterval()

Definition:

The clearInterval() method is used to stop the repeated execution set by setInterval().

Syntax:

clearInterval(intervalID)

Example:


let counter = 0;
let timer = setInterval(() => {
  console.log("Counter:", counter);
  counter++;
  if (counter === 3) {
    clearInterval(timer);
    console.log("Interval stopped!");
  }
}, 1000);

// Output:
// Counter: 0
// Counter: 1
// Counter: 2
// Interval stopped!
