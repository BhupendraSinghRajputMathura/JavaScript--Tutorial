üìò Data Types in JavaScript

JavaScript provides different data types to hold different kinds of values. There are two main categories:

  • Primitive Data Types (7)
  • Non-Primitive Data Types (Objects)

üü¢ Primitive Data Types

  • String: Textual data ‚Üí "Hello, World!"
  • Number: Integers or floating points ‚Üí 42, 3.14
  • Boolean: Logical values ‚Üí true, false
  • Null: Represents no value ‚Üí null
  • Undefined: Declared but not assigned ‚Üí undefined
  • Symbol: Unique identifier ‚Üí Symbol('id')
  • BigInt: Large integers ‚Üí 9007199254740991n

üü£ Non-Primitive Data Types (Reference Types)

  • Object: Key-value pairs ‚Üí {name: "John", age: 30}
  • Array: Ordered collection ‚Üí [1, 2, 3, 4, 5]
  • Function: Block of reusable code ‚Üí function greet() { console.log("Hello!"); }
  • Date: Represents date/time ‚Üí new Date()
  • RegExp: Pattern matching ‚Üí /ab+c/

üìù Important Notes

  • Primitive data types are immutable.
  • Non-primitive types (objects) are mutable.
  • JavaScript is dynamically typed ‚Äî type is determined at runtime.

Example:


let name = "Alice"; // String
let age = 30; // Number
let isStudent = false; // Boolean
let address = null; // Null
let phoneNumber; // Undefined
let uniqueId = Symbol('id'); // Symbol
let bigNumber = 9007199254740991n; // BigInt
let person = { name: "Bob", age: 25 }; // Object
let numbers = [1, 2, 3, 4, 5]; // Array
function greet() { console.log("Hello!"); } // Function
let currentDate = new Date(); // Date
let pattern = /ab+c/; // RegExp

console.log(typeof [1, 2, 3]); // "object"
console.log(typeof function() {}); // "function"

üí° Notes on String Representation and Template Literals

  • Strings can be defined using "", '', or ``.
  • Backticks are used for template literals to embed variables.

Examples:


// Using single and double quotes
let greeting1 = "Hello, World!";
let greeting2 = 'Hello, World!';

// Double quotes can hold single quotes easily
let message1 = "It's a beautiful day!";
let message2 = 'It\'s a beautiful day!';

// Template literals
let nameUser = "Aadi";
let greeting = `Hello, ${nameUser}!`;

// Multi-line string
let multiLine = `This is a string
that spans multiple lines.`;

// Expressions inside template literals
let a = 5, b = 10;
let result = `The sum of ${a} and ${b} is ${a + b}.`;
