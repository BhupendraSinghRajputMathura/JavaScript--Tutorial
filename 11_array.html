Arrays in JavaScript

An Array is a special variable that can hold more than one value at a time. Arrays are used to store multiple values in a single variable.

Example


const fruits = ["Apple", "Banana", "Mango"];
let numbers = [1, 2, 3, 4, 5];
var mixed = ["Hello", 42, true, null];

Slice vs Splice


// slice() â€“ returns a new array, original remains unchanged
const fruits = ["Apple", "Banana", "Mango", "Orange"];
const citrus = fruits.slice(1, 3); 
console.log(citrus); // ["Banana", "Mango"]

// splice() â€“ modifies original array
fruits.splice(1, 2, "Grapes");
console.log(fruits); // ["Apple", "Grapes", "Orange"]

Advanced Array Features


// 1. Spread Operator (...)
const arr1 = [1,2,3];
const arr2 = [4,5,6];
const merged = [...arr1, ...arr2];
console.log(merged); // [1,2,3,4,5,6]

// 2. flat() â€“ Flattens nested arrays
const nested = [1, [2, [3, [4]]]];
console.log(nested.flat(Infinity)); // [1,2,3,4]

// 3. Array.from() â€“ Converts string or iterable to array
console.log(Array.from("BHUPENDRA")); 
// ['B','H','U','P','E','N','D','R','A']

// 4. Array.of() â€“ Creates array from individual values
const nums = Array.of(10, 20, 30, 40);
console.log(nums); // [10, 20, 30, 40]
