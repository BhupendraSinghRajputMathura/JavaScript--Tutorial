Immediately Invoked Function Expression (IIFE)

An IIFE is a function that runs immediately after it is defined. It helps in avoiding global scope pollution and creating a private scope.

Syntax

(function() {
    // Code to be executed immediately
})();

Example

(function() {
    console.log("IIFE executed!");
})();

Use Cases

  • Creating a private scope
  • Initializing variables safely
  • Preventing global namespace pollution
