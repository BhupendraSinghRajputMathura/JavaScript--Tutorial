DOM Level 2 Event Handler (Modern Method)

üìò Definition

DOM Level 2 Event Handling is a modern method in JavaScript for handling events using the addEventListener() and removeEventListener() methods. It allows developers to attach multiple event listeners to the same element without overwriting previous ones, providing more control and flexibility.

üìÑ Syntax


element.addEventListener(event, function, useCapture);
element.removeEventListener(event, function, useCapture);

üîπ Parameters:

  • event: The name of the event (e.g., "click", "mouseover").
  • function: The function that will be executed when the event occurs.
  • useCapture (optional): A boolean value that specifies whether the event should be captured in the capturing phase or bubbling phase. Default is false.

‚öôÔ∏è Types of DOM Level 2 Event Handling

  1. Mouse Events (e.g., click, dblclick, mouseover, mouseout, mousedown, mouseup)
  2. Keyboard Events (e.g., keydown, keyup, keypress)
  3. Form Events (e.g., submit, change, focus, blur)
  4. Window Events (e.g., load, resize, scroll)

üñ±Ô∏è 1. Mouse Events

Example:


<button id="btn">Click Me</button>

<script>
  const button = document.getElementById("btn");

  function showMessage() {
    alert("Button clicked!");
  }

  button.addEventListener("click", showMessage);
</script>

Explanation: Here, the click event is attached to the button using addEventListener().

‚å®Ô∏è 2. Keyboard Events

Example:


<input type="text" id="name" placeholder="Type something...">

<script>
  const input = document.getElementById("name");

  input.addEventListener("keydown", function(event) {
    console.log("Key pressed:", event.key);
  });
</script>

Explanation: The keydown event triggers when any key is pressed inside the input field.

üßæ 3. Form Events

Example:


<form id="myForm">
  <input type="text" placeholder="Enter name">
  <button type="submit">Submit</button>
</form>

<script>
  const form = document.getElementById("myForm");

  form.addEventListener("submit", function(event) {
    event.preventDefault(); // prevents form submission
    alert("Form submitted successfully!");
  });
</script>

Explanation: The submit event is handled without reloading the page using preventDefault().

ü™ü 4. Window Events

Example:


<script>
  window.addEventListener("resize", function() {
    console.log("Window resized. Current width:", window.innerWidth);
  });

  window.addEventListener("load", function() {
    console.log("Page fully loaded!");
  });
</script>

Explanation: The resize event runs when the browser window size changes, and load fires when the page finishes loading.

üß© Removing Event Listeners

You can remove an attached event handler using removeEventListener().


<button id="removeBtn">Click Me</button>

<script>
  const removeBtn = document.getElementById("removeBtn");

  function handleClick() {
    alert("Event listener removed after one click!");
    removeBtn.removeEventListener("click", handleClick);
  }

  removeBtn.addEventListener("click", handleClick);
</script>

‚úÖ Advantages of Modern Event Handling

  • Allows multiple event handlers on the same element.
  • Separates HTML from JavaScript (clean code).
  • Supports both capturing and bubbling phases.
  • Can easily remove specific event handlers.
