Higher Order Functions in JavaScript

Higher-order functions are functions that can take other functions as arguments or return them as output. They are a fundamental part of functional programming, enabling cleaner, reusable, and more abstract code.

Examples

1️⃣ Functions as First-Class Citizens

function greet(name) {
  return "Hello, " + name;
}

function processUserInput(callback) {
  var name = prompt("Please enter your name.");
  alert(callback(name));
}

processUserInput(greet);

2️⃣ Array Methods

const numbers = [1, 2, 3, 4, 5];

const doubled = numbers.map(num => num * 2);
const filtered = numbers.filter(num => num > 2);

3️⃣ Returning Functions

function makeCounter() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}

const counter = makeCounter();
console.log(counter()); // 1
console.log(counter()); // 2
console.log(counter()); // 3

Common Higher Order Functions

  • forEach() – Executes a function once per array element.
  • filter() – Filters elements in an array based on a condition.
  • map() – Transforms each element in an array.
  • reduce() – Reduces an array to a single cumulative value.

.forEach()

forEach() executes a function for each array element but does not return anything (always undefined). 

const arr = [1, 2, 3, 4, 5];
arr.forEach(element => console.log(element));
const fruits = ['apple', 'banana', 'cherry'];
fruits.forEach((fruit, index) => console.log(index + ': ' + fruit));

.filter()

The filter() method creates a new array with elements that pass a test. 

const numbers = [1, 2, 3, 4, 5];
const even = numbers.filter(num => num % 2 === 0);
console.log(even); // [2, 4]

.map()

The map() method returns a new array with the results of calling a function for every element. 

const numbers = [1, 2, 3, 4, 5];
const squared = numbers.map(num => num * num);
console.log(squared); // [1, 4, 9, 16, 25]

.reduce()

The reduce() method executes a reducer function on each element of the array, resulting in a single output value. 

const numbers = [1, 2, 3, 4, 5];
const sum = numbers.reduce((acc, cur) => acc + cur, 0);
console.log(sum); // 15

Chaining Methods

const result = [1, 2, 3, 4, 5]
  .map(num => num * 2)
  .filter(num => num > 5);
console.log(result); // [6, 8, 10]

Comparison Table

  • map() – Returns new transformed array ✅
  • filter() – Returns filtered array ✅
  • reduce() – Returns single value ✅
  • forEach() – No return ❌
