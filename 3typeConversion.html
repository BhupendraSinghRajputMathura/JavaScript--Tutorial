Data-Type Conversion in JavaScript

This page demonstrates various data-type conversions in JavaScript, including both explicit and implicit conversions.

  • String to Number
  • Number to String
  • Boolean to Number
  • Number to Boolean
  • String to Boolean
  • Implicit Conversions
  • Special Cases and Edge Cases
  1. String to Number: Using the `Number()` function or unary `+` operator.
  2. Number to String: Using the `String()` function or `toString()` method.
  3. Boolean to Number: `true` converts to `1`, `false` converts to `0`.
  4. Number to Boolean: `0`, `NaN`, `null`, `undefined`, and `""` convert to `false`; all other values convert to `true`.
  5. String to Boolean: Non-empty strings convert to `true`, empty strings convert to `false`.
  6. Implicit Conversions: Occur during operations like addition, subtraction, etc.
  7. Special Cases and Edge Cases: Handling of `null`, `undefined`, `NaN`, arrays, objects, and functions.
  8. Equality vs. Strict Equality: Differences between `==` and `===` in type coercion.
  9. Object to Primitive Conversion: How objects are converted to primitive types using `valueOf()` and `toString()`.
  10. Using `parseInt()` and `parseFloat()`: Converting strings to numbers with specific parsing rules.
  11. Hexadecimal, Binary, and Octal Strings: Converting strings with different bases to numbers.
  12. Handling Whitespace in Strings: How leading and trailing whitespace affects conversions.
  13. Conversion of Special Numeric Values: Handling `Infinity`, `-Infinity`, and `NaN`.
  14. Conversion of Date Objects: How `Date` objects convert to strings and numbers.
  15. Conversion of Symbol Type: Understanding that `Symbol` cannot be converted to a number or string directly.
  16. Using Template Literals for Conversion: How template literals can implicitly convert values to strings.

// String to Number

let str = "123";
let num = Number(str);
console.log(num); // 123
console.log(typeof num); // "number"

// Number to String

let n = 456;
let s = String(n);
console.log(s); // "456"
console.log(typeof s); // "string"

// Boolean to Number

let boolTrue = true;
let boolFalse = false;
console.log(Number(boolTrue)); // 1
console.log(Number(boolFalse)); // 0

// Number to Boolean

let zero = 0;
let nonZero = 42;
console.log(Boolean(zero)); // false
console.log(Boolean(nonZero)); // true

// String to Boolean

let emptyStr = "";
let nonEmptyStr = "Hello";
console.log(Boolean(emptyStr)); // false
console.log(Boolean(nonEmptyStr)); // true

// Implicit Conversion

let implicitNum = "5" * 2; // String to Number
console.log(implicitNum); // 10
let implicitStr = 5 + " apples"; // Number to String
console.log(implicitStr); // "5 apples"
console.log(typeof implicitStr); // "string"
let implicitBool = 1 + true; // Boolean to Number
console.log(implicitBool); // 2
console.log(typeof implicitBool); // "number"
let implicitBoolStr = "Value: " + false; // Boolean to String
console.log(implicitBoolStr); // "Value: false"
console.log(typeof implicitBoolStr); // "string"
