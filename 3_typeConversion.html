Data-Type Conversion in JavaScript

This page demonstrates various data-type conversions in JavaScript, including both explicit and implicit conversions.

  • String to Number
  • Number to String
  • Boolean to Number
  • Number to Boolean
  • String to Boolean
  • Implicit Conversions
  • Special and Edge Cases

Types of Conversions

  1. String to Number: Using Number() or unary + operator.
  2. Number to String: Using String() or toString() method.
  3. Boolean to Number: true → 1, false → 0.
  4. Number to Boolean: Falsy values: 0, NaN, null, undefined, ""; everything else is true.
  5. String to Boolean: Empty string → false, non-empty → true.
  6. Implicit Conversions: Automatic type conversions during operations.
  7. Special Cases: null, undefined, NaN, arrays, and objects.

Example Code


// String to Number
let str = "123";
let num = Number(str);
console.log(num); // 123

// Number to String
let n = 456;
let s = String(n);
console.log(s); // "456"

// Boolean to Number
console.log(Number(true));  // 1
console.log(Number(false)); // 0

// Number to Boolean
console.log(Boolean(0));  // false
console.log(Boolean(42)); // true

// String to Boolean
console.log(Boolean(""));      // false
console.log(Boolean("Hello")); // true

// Implicit Conversion
console.log("5" * 2);     // 10
console.log(5 + " apples"); // "5 apples"

// Special Cases
console.log(Number("")); // 0
console.log(Boolean([])); // true
console.log(String({}));  // "[object Object]"
