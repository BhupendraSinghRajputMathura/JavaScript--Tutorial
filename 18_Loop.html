Loops in JavaScript

Loops allow you to execute a block of code multiple times, making them essential for automation and efficient programming.

Types of Loops

  • for loop – Used when the number of iterations is known.
  • while loop – Used when the number of iterations depends on a condition.
  • do...while loop – Executes code at least once before checking the condition.
  • for...in loop – Iterates over the properties of an object.
  • for...of loop – Iterates over iterable objects like arrays or strings.

Examples

for loop

for (let i = 0; i < 5; i++) {
    console.log(i);
}

while loop

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

do...while loop

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

for...in loop

Used to iterate over the keys of an object:

const obj = {a: 1, b: 2, c: 3};
for (const key in obj) {
    console.log(key + ": " + obj[key]);
}

for...of loop

Used to iterate over values in iterable objects like arrays or strings:

const arr = [10, 20, 30];
for (const value of arr) {
    console.log(value);
}

new Map() in JavaScript

Map is a collection of key-value pairs where both key and value can be of any type.

const myMap = new Map();
myMap.set('name', 'John');
myMap.set('age', 30);
myMap.set(1, 'one');

console.log(myMap);
console.log(myMap.get('name')); // Output: John
console.log(myMap.size); // Output: 3
myMap.delete(1);
console.log(myMap.has('age')); // true
myMap.clear();
