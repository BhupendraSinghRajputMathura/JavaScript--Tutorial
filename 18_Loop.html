Loops in JavaScript

Loops are used to execute a block of code repeatedly until a specified condition is met. JavaScript supports several types of loops:

  • for loop: Used when the number of iterations is known.
  • while loop: Used when the number of iterations is not known and depends on a condition.
  • do...while loop: Similar to the while loop, but guarantees that the code block will be executed at least once.
  • for...in loop: Used to iterate over the properties of an object.
  • for...of loop: Used to iterate over iterable objects like arrays, strings, etc.

Examples

for loop

for (let i = 0; i < 5; i++) {
    console.log(i);
}

while loop

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

do...while loop

let i = 0;   
do {
    console.log(i);
    i++;
} while (i < 5);

for...in loop

Used to iterate over the properties of an object for keys.

const obj = {a: 1, b: 2, c: 3};
for (const key in obj) {
    console.log(key + ": " + obj[key]);
}

for...of loop

Used to iterate over iterable objects like arrays, strings, etc for values.

const arr = [10, 20, 30];
for (const value of arr) {
    console.log(value);  
}

for...in vs for...of

The for...in loop iterates over the enumerable properties of an object, while the for...of loop iterates over the values of an iterable object like an array or string. Use for...in for objects and for...of for arrays and other iterable collections.

new Map()

The Map object in JavaScript is a collection of key-value pairs where both keys and values can be of any data type. It maintains the order of insertion and allows for efficient retrieval, addition, and deletion of elements.

Example:

const myMap = new Map();
myMap.set('name', 'John');
myMap.set('age', 30);
myMap.set(1, 'one');
myMap.set('name', 'John'); // Duplicate key, will update the value

console.log(myMap); // Output: Map(3) {'name' => 'John', 'age' => 30, 1 => 'one'}   
console.log(myMap.get('name')); // Output: John
console.log(myMap.size); // Output: 3
myMap.delete(1); // Removes the key-value pair with key 1
console.log(myMap.has('age')); // Output: true
myMap.clear(); // Removes all key-value pairs

new Map() operation

  • set(key, value): Adds or updates an element with the specified key and value.
  • get(key): Retrieves the value associated with the specified key.
  • has(key): Checks if an element with the specified key exists in the Map.
  • delete(key): Removes the element with the specified key from the Map.
  • clear(): Removes all elements from the Map.
  • size: Returns the number of elements in the Map.
