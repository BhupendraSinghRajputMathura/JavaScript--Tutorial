JavaScript Number

Understanding numerical operations and methods in JavaScript

JavaScript has only one type of number. Numbers can be written with or without decimals:

Example

let x = 3.14;
let y = 3;

Common Number Methods

  • Number.isInteger() → Checks if the value is an integer.
  • Number.parseFloat() → Converts a string to a floating-point number.
  • Number.parseInt() → Converts a string to an integer.
  • toString() → Converts a number to a string.
  • toFixed() → Formats a number with fixed decimals.
  • toPrecision() → Formats a number to a specified precision.
  • valueOf() → Returns the primitive value of a number.

Example Usage

let x = 123;
let y = 123.456;

console.log(Number.isInteger(x));        // true
console.log(Number.isInteger(y));        // false
console.log(Number.parseFloat("123.456")); // 123.456
console.log(Number.parseInt("123.456"));   // 123
console.log(x.toString());               // "123"
console.log(y.toFixed(2));               // "123.46"
console.log(y.toPrecision(4));           // "123.5"
console.log(x.valueOf());                // 123
console.log(y.valueOf());                // 123.456

These methods help in handling numbers effectively — whether you’re formatting, converting, or validating numerical values in your JavaScript programs.
