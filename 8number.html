JavaScript Number

JavaScript has only one type of number. Numbers can be written with or without decimals:

Example

let x = 3.14

let y = 3

Methodes

Number.isInteger()

Number.parseFloat()

Number.parseInt()

Number.toString()

Number.toFixed()

Number.toPrecision()

Number.valueOf()

Example


    let x = 123;
    let y = 123.456;

console.log(Number.isInteger(x)); // true

console.log(Number.isInteger(y)); // false

console.log(Number.parseFloat("123.456")); // 123.456

console.log(Number.parseInt("123.456")); // 123

console.log(x.toString()); // "123"

console.log(y.toFixed(2)); // "123.46"

console.log(y.toPrecision(4)); // "123.5"

console.log(x.valueOf()); // 123

console.log(y.valueOf()); // 123.456
