📘 Variables in JavaScript

A variable in JavaScript is a container used to store data values. Variables allow programs to store, modify, and reuse data.

🟢 Ways to Declare Variables

There are three ways to declare variables in JavaScript:

  • var
  • let
  • const

🔹 Important Notes

  • Variables declared with var or let can be reassigned.
  • Variables declared with const cannot be reassigned.

Example:


const panCard = "1234-5678-9012-3456";
let age = 25;
var name = "John Doe";
  1. panCard is a constant variable (cannot be changed).
  2. age can be reassigned.
  3. name can also be reassigned.

✅ Use const for values that do not change, and let for variables that might. Avoid var in modern JavaScript due to potential scope issues.

💡 Difference between var, let, and const

🔸 Scope

  • var: Function-scoped — accessible throughout a function.
  • let: Block-scoped — accessible only within the block {}.
  • const: Block-scoped — accessible only within the block {}.

🔸 Reassignment

  • var: Can be reassigned.
  • let: Can be reassigned.
  • const: Cannot be reassigned.

🔸 Hoisting

  • var: Hoisted and initialized with undefined.
  • let: Hoisted but not initialized (temporal dead zone).
  • const: Hoisted but not initialized (temporal dead zone).

🔸 Use Cases

  • var: Avoid using in modern code.
  • let: Use for variables that may change.
  • const: Use for constants or references that should not change.

Example:


var x = 10;   // function-scoped, can be reassigned
let y = 20;   // block-scoped, can be reassigned
const z = 30; // block-scoped, cannot be reassigned

🖥️ JavaScript Console Methods

The console object provides methods for developers to log, debug, and monitor JavaScript code.

  • console.log() → Print messages or values to the console.
  • console.error() → Display error messages.
  • console.warn() → Display warning messages.
  • console.table() → Show data in table format.
  • console.info() → Display informational messages.
  • console.debug() → Show debug messages.
  • console.clear() → Clear the console output.
  • console.time() / console.timeEnd() → Measure code execution time.
  • console.group() / console.groupEnd() → Group related log messages.
  • console.trace() → Display the stack trace.
