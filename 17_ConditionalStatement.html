Conditional Statements in JavaScript

Conditional statements are used to perform different actions based on conditions. They control the flow of execution in a program by executing certain blocks of code only when specific conditions are met.

Types of Conditional Statements

  • if statement: Executes a block of code if a condition is true.
  • if...else statement: Executes one block if a condition is true, another if false.
  • else if statement: Tests multiple conditions sequentially.
  • switch statement: Compares an expression against multiple values.

Examples

if Statement

if (condition) {
    // code executes if condition is true
}

if...else Statement

if (condition) {
    // executes if condition is true
} else {
    // executes if condition is false
}

else if Statement

if (condition1) {
    // executes if condition1 is true
} else if (condition2) {
    // executes if condition2 is true
} else {
    // executes if all are false
}

switch Statement

switch (expression) {
    case value1:
        // code executes if expression === value1
        break;
    case value2:
        // code executes if expression === value2
        break;
    default:
        // code executes if no case matches
}

Ternary Operator

The ternary operator provides a shorthand way of writing if...else. It takes three operands: a condition, a value if true, and a value if false. 

condition ? valueIfTrue : valueIfFalse;

Example:

let age = 20;
let status = (age < 18) ? "minor" : "adult";
console.log(status); // Output: adult

Best Practices

  • Write simple, clear conditions for better readability.
  • Use switch when checking multiple discrete values.
  • Avoid deeply nested if statements; refactor with functions or logical operators.
  • Always include a default case in switch statements.

Conclusion

Conditional statements are fundamental to decision-making in JavaScript. They help create dynamic, responsive, and logical programs that react to user inputs and changing data.
