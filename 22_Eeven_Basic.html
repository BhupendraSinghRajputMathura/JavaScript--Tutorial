ðŸ”¹ Types of Event Handlers

1. Inline Event Handler

Defined directly in the HTML element using attributes like onclick.

<button onclick="alert('Button Clicked!')">Click Me</button>

2. DOM Level 0 Event Handler (Classic Method)

Assigned directly to a DOM element property in JavaScript.

<button id="btn">Click Me</button>

<script>
  const btn = document.getElementById('btn');
  btn.onclick = function() {
    alert('Button clicked!');
  };
</script>

3. DOM Level 2 Event Handler (Modern Method)

Uses addEventListener() to attach multiple handlers.

<button id="myButton">Click Me</button>

<script>
  const myButton = document.getElementById('myButton');
  myButton.addEventListener('click', function() {
    alert('Clicked using addEventListener!');
  });
</script>

ðŸ”¹ Removing an Event Listener

function showMessage() {
  alert('Hello!');
}

myButton.addEventListener('click', showMessage);

// To remove:
myButton.removeEventListener('click', showMessage);
