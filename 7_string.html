String

A string is a sequence of characters used to represent text. In programming, strings are typically enclosed in quotes, either single (' ') or double (" ") and backtick (` `).

  2. ( ` ` )(backtick) - Template literals (multi-line + interpolation + advanced features)
  1. Single quotes: 'Hello, World!'
  2. Double quotes: "Hello, World!"
  3. Backticks (Template literals): `Hello, World!`

Strings can contain letters, numbers, symbols, and spaces. They are commonly used for storing and manipulating text data in applications.

Example of a string in JavaScript:

let greeting = "Hello, World!";

Strings can be concatenated, sliced, and manipulated using various methods provided by programming languages.

For example, in JavaScript, you can concatenate strings using the + operator:

let firstName = "John";  
let lastName = "Doe";  
let fullName = firstName + " " + lastName; // "John Doe"

Strings are immutable in many programming languages, meaning that once a string is created, it cannot be changed. However, you can create new strings based on existing ones.

Common string methods in JavaScript include:

  • length: Returns the length of the string.
  • toUpperCase(): Converts the string to uppercase.
  • toLowerCase(): Converts the string to lowercase.
  • substring(start, end): Extracts a portion of the string.
  • indexOf(substring): Returns the index of the first occurrence of a substring.
  • replace(oldSubstring, newSubstring): Replaces occurrences of a substring with a new substring.
  • split(separator): Splits the string into an array of substrings based on a specified separator.
  • trim(): Removes whitespace from both ends of the string.
