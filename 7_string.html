Strings in JavaScript

Understanding text handling and manipulation in JavaScript

A string is a sequence of characters used to represent text. In JavaScript, strings can be defined using 'single', "double", or `backtick` quotes.

Types of String Declaration

  1. Single quotes: 'Hello, World!'
  2. Double quotes: "Hello, World!"
  3. Backticks (Template literals): `Hello, World!`

Basic Example

let greeting = "Hello, World!";
console.log(greeting); // Output: Hello, World!

String Concatenation

let firstName = "John";
let lastName = "Doe";
let fullName = firstName + " " + lastName;
console.log(fullName); // John Doe

Template Literals Example

let name = "Alice";
let age = 25;
let info = `My name is ${name} and I am ${age} years old.`;
console.log(info);

Common String Methods

  • length → Returns the string length.
  • toUpperCase() → Converts to uppercase.
  • toLowerCase() → Converts to lowercase.
  • substring(start, end) → Extracts part of the string.
  • indexOf(value) → Finds the first occurrence of a substring.
  • replace(old, new) → Replaces text in the string.
  • split(separator) → Splits string into an array.
  • trim() → Removes whitespace from both ends.

Live Example (Run in Console)

let example = "  Hello, JavaScript!  ";
console.log(example.trim());
console.log(example.toUpperCase());
console.log(example.replace("JavaScript", "World"));
console.log(example.split(" "));
