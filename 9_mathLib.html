JavaScript Math Library

The Math object in JavaScript provides mathematical constants and functions. It is not a constructor and can be used directly to perform various calculations.

Common Methods

  • Math.abs(x) - Returns the absolute value of x.
  • Math.acos(x) - Returns the arccosine of x, in radians.
  • Math.asin(x) - Returns the arcsine of x, in radians.
  • Math.atan(x) - Returns the arctangent of x, in radians.
  • Math.atan2(y, x) - Returns the arctangent of the quotient of its arguments.
  • Math.ceil(x) - Rounds x up to the nearest integer.
  • Math.cos(x) - Returns the cosine of x (x is in radians).
  • Math.exp(x) - Returns ex, where e is Euler's number.
  • Math.floor(x) - Rounds x down to the nearest integer.
  • Math.log(x) - Returns the natural logarithm (base e) of x.
  • Math.max(...args) - Returns the largest of zero or more numbers.
  • Math.min(...args) - Returns the smallest of zero or more numbers.
  • Math.pow(base, exponent) - Returns base raised to the power of exponent.
  • Math.random() - Returns a pseudo-random number between 0 and 1.
  • Math.round(x) - Rounds x to the nearest integer.
  • Math.sin(x) - Returns the sine of x (x is in radians).
  • Math.sqrt(x) - Returns the square root of x.
  • Math.tan(x) - Returns the tangent of x (x is in radians).

Example


let num = -5.7;
let absoluteValue = Math.abs(num); // 5.7
let roundedValue = Math.round(num); // -6
let randomValue = Math.random(); // Random number between 0 and 1
let powerValue = Math.pow(2, 3); // 8
let squareRoot = Math.sqrt(16); // 4

console.log("Absolute Value:", absoluteValue);
console.log("Rounded Value:", roundedValue);
console.log("Random Value:", randomValue);
console.log("2 to the power of 3:", powerValue);
console.log("Square Root of 16:", squareRoot);

The Math object is an essential part of JavaScript for performing accurate and efficient mathematical operations in any web application.
