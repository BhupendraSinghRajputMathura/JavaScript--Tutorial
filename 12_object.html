Objects in JavaScript

In JavaScript, an object is a complex data type that allows you to store collections of data and more complex entities. Objects are used to represent real-world entities and can contain properties (key-value pairs) and methods (functions associated with the object).

Creating Objects

There are several ways to create objects in JavaScript:

1. Object Literal

let person = {
    name: "Alice",
    age: 25,
    greet: function() {
        console.log("Hello, my name is " + this.name);
    }
};

2. Using the Object Constructor

let car = new Object();
car.make = "Toyota";
car.model = "Camry";
car.year = 2020;

3. Using a Constructor Function

function Dog(name, breed) {
    this.name = name;
    this.breed = breed;
    this.bark = function() {
        console.log("Woof! Woof!");
    };
}   
let myDog = new Dog("Buddy", "Golden Retriever");

Accessing Object Properties and Methods

You can access object properties and methods using dot notation or bracket notation:

// Using dot notation
console.log(person.name); // Output: Alice
person.greet(); // Output: Hello, my name is Alice
// Using bracket notation
console.log(car["make"]); // Output: Toyota
console.log(myDog["breed"]); // Output: Golden Retriever
myDog.bark(); // Output: Woof! Woof!

Modifying Objects

You can add, update, or delete properties and methods of an object:

// Adding a new property
person.height = 170; // in cm
// Updating an existing property
car.year = 2021;
// Deleting a property
delete myDog.breed;

Iterating Over Object Properties

You can use a for...in loop to iterate over the properties of an object:

for (let key in person) {
    console.log(key + ": " + person[key]);
}

Object Methods

JavaScript provides several built-in methods for working with objects, such as:

  • Object.keys(obj): Returns an array of the object's property names.
  • Object.values(obj): Returns an array of the object's property values.
  • Object.entries(obj): Returns an array of the object's key-value pairs.
  • Object.assign(target, ...sources): Copies properties from source objects to a target object.
  • JSON.stringify(obj): Converts a JavaScript object into a JSON string.
  • JSON.parse(jsonString): Parses a JSON string and converts it into a JavaScript object.

Understanding how to create and manipulate objects is essential for effective JavaScript programming, as they are widely used in various applications and frameworks.

Object Method Examples

// Creating an object using object literal
let student = {
    name: "Bob",
    age: 22,
    courses: ["Math", "Science", "History"],
    introduce: function() {
        console.log("Hi, I'm " + this.name + " and I'm " + this.age + " years old.");
    }
};

// Accessing properties and methods
console.log(student.name); // Output: Bob
student.introduce(); // Output: Hi, I'm Bob and I'm 22 years old.
console.log(student.courses[1]); // Output: Science

// Modifying the object
student.age = 23; // Updating age
student.courses.push("Art"); // Adding a new course
console.log(student.courses); // Output: ["Math", "Science", "History", "Art"]
student.introduce(); // Output: Hi, I'm Bob and I'm 23 years old.

Important

constructor methods: Provide singleton

Object.create(): Create a new object with the specified prototype object and properties.

Object.defineProperty(): Define a new property directly on an object, or modify an existing property on an object, and return the object.

Object.freeze(): Freeze an object. A frozen object can no longer be modified.

Object.seal(): Seal an object, preventing new properties from being added to it and marking all existing properties as non-configurable.

Object.getPrototypeOf(): Returns the prototype (i.e., the internal [[Prototype]] property) of the specified object.

Object.setPrototypeOf(): Sets the prototype (i.e., the internal [[Prototype]] property) of a specified object to another object or null.

Accessing Object

There are two ways to access the properties of an object: dot notation and bracket notation.

Dot Notation: This is the most common way to access object properties. You simply use the object's name followed by a dot and the property name.

let person = {       
    name: "Alice",
    age: 25
};
console.log(person.name); // Output: Alice
console.log(person.age);  // Output: 25

Bracket Notation: This method is useful when the property name is dynamic or not a valid identifier (e.g., contains spaces or special characters). You use the object's name followed by brackets containing the property name as a string.

let person = {       
    "first name": "Alice",
    age: 25
};
console.log(person["first name"]); // Output: Alice
console.log(person["age"]);        // Output: 25

Declare and Accessing Symbol in object

let sym1 = Symbol("id");

let user = {
    [sym1]: 101,
    [sym2]: 102,
    name: "Charlie"
};

console.log(user[sym2]); // Output: 102
console.log(user.name); // Output: Charlie

let obj2 = {b: 3, c: 4};

console.log(mergedObj);
