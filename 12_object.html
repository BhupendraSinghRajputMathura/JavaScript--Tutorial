Objects in JavaScript

Comprehensive Guide with Examples and Explanations

What is an Object?

In JavaScript, an object is a complex data type that allows you to store collections of data and more complex entities.

Objects contain properties (key-value pairs) and methods (functions associated with the object).

Creating Objects

1. Object Literal

let person = {
    name: "Alice",
    age: 25,
    greet: function() {
        console.log("Hello, my name is " + this.name);
    }
};

2. Object Constructor

let car = new Object();
car.make = "Toyota";
car.model = "Camry";
car.year = 2020;

3. Constructor Function

function Dog(name, breed) {
    this.name = name;
    this.breed = breed;
    this.bark = function() {
        console.log("Woof! Woof!");
    };
}
let myDog = new Dog("Buddy", "Golden Retriever");

Accessing Object Properties and Methods

console.log(person.name); // "Alice"
person.greet(); // Hello, my name is Alice
console.log(car["make"]); // "Toyota"
myDog.bark(); // "Woof! Woof!"

Modifying Objects

person.height = 170;
car.year = 2021;
delete myDog.breed;

Iterating Over Object Properties

for (let key in person) {
    console.log(key + ": " + person[key]);
}

Object Destructuring

let student = { name: "Bob", age: 23, course: "Math" };
