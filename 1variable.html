Variable in JavaScript

A variable is a container for storing data values.

there are 3 ways to declare a variable in JavaScript:
  • var
  • let
  • const

note:

  • Variables declared with var or let can be reassigned.
  • Variables declared with const cannot be reassigned.
Example:
  • const panCard = "1234-5678-9012-3456";
  • let age = 25;
  • var name = "John Doe";

  1. In the above example, panCard is a constant variable that cannot be changed, age is a variable that can be reassigned, and name is a variable that can also be reassigned.
  2. Prefer using const for variables that should not change, and let for variables that may change. Avoid using var to prevent scope-related issues.
