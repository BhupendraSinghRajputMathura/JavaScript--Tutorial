Understanding Objects in JavaScript

  1. Definition:

    Collection of key-value pairs where each key is distinct.

  2. Declaration:

    const obj1 = {id:1, age:21, name:"aadi"};

    OR

    const obj1 = {
  id:1,
  age:21,
  name:"aadi"
};

  3. Accessing:

    const obj1 = {
  id:1,
  age:21,
  name:"aadi"
};

console.log(obj1.id); // 1
console.log(obj1["id"]); // 1

  4. Overwriting:

    obj1.id = 2;
obj1.age = 22;
obj1.name = "Singh";

  5. Object.freeze(object_name)

    Locks the object â€” makes it immutable.

    Object.freeze(obj1);
obj1.id = 5; // won't change

  6. Object.keys(object_name)

    Returns all keys.

    Object.keys(obj1); // ['id','age','name']

  7. Object.values(object_name)

    Returns all values.

    Object.values(obj1); // [1, 21, 'aadi']

  8. Object.entries(object_name)

    Returns key-value pairs.

    Object.entries(obj1);
// [['id',1], ['age',21], ['name','aadi']]

  9. Spread Operator (...)

    • Copy Object
      • const copy = {...obj1};
    • Merging Objects
      • const merged = {...obj1, ...obj2};
    • Override
      • const updated = {...obj1, age:25};
    • Nested Object (Shallow Copy Issue)
      • const student = {
  name: "Raj",
  address: { city: "Mathura" }
};
const newStudent = {...student};
newStudent.address.city = "Delhi";

  10. De-structuring of Object:

    const obj = { id:1, age:21, name:"Aadi" };
const { id, age, name } = obj;
console.log(name); // Aadi
